# Psicologa Annachiara Lorusso - Sito vetrina

Repository di lavoro per progettazione e sviluppo di un sito vetrina.

## Stack deciso
- Astro (sito statico)
- Tailwind CSS (frontend)

## Obiettivo
Costruire una base solida per:
- analizzare l'intervista cliente
- definire strategia e struttura del sito
- iterare sui testi (copy)
- arrivare a una prima versione del sito

## Flusso di lavoro
1. Inserire il materiale cliente (PDF intervista) in docs/brief/input.
2. Estrarre i punti chiave in docs/brief/intervista-cliente.md.
3. Elaborare direzione strategica in docs/strategia/idee-sito-vetrina.md.
4. Lavorare sui testi in docs/contenuti/copy-bozze.md.

## Struttura cartelle
- docs/brief: materiale di partenza e analisi intervista
- docs/strategia: idee, direzione, architettura dei contenuti
- docs/contenuti: bozze testi e revisioni
- assets: risorse statiche (immagini, allegati non sensibili)

## Prossimo step
Consolidare architettura pagine e passare alla bozza wireframe/copy per Home, Chi sono, Servizi, Aree di intervento, FAQ, Contatti.
